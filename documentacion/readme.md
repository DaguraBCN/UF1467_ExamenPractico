## 📦 Instalación

### Requisitos

- Python 3.10+
- pip
- Git
- (opcional) Docker

### Pasos

```bash
git clone https://github.com/tuusuario/tu-repositorio.git
cd tu-repositorio
python -m venv env
source env/bin/activate  # En Windows: env\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```

## ▶️ Uso

Abre tu navegador y accede a:

http://localhost:8000/
